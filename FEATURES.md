# Features

**ERMwatch v1.1** – Simple E-Resources Monitor for Libraries

### Monitoring
- Automatic checks every 10 minutes
- Supports all types of URLs (databases, e-journals, e-books, websites, etc.)
- Smart status detection with failure reasons

### User Interface
- Clean dashboard with live statistics
- Color-coded status (Green = Working, Red blinking = Down)
- Desktop notifications for downtime
- Search functionality
- Dark/Light theme toggle
- Multi-language support (8 languages)

### Management
- Add, edit, delete resources
- Bulk import from CSV
- Export resources to CSV
- Download import template

### Reports
- Professional PDF reports with proper formatting
- Includes status, last checked time, and failure reasons

### Practical Benefits
- Zero installation
- Runs completely offline
- No data collection or telemetry
- Automatic desktop shortcut
- Basic tamper protection

Built for librarians who need a simple and reliable monitoring tool.
