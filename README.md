# ERMwatch v1.1

**Free Standalone E-Resources Monitor for Libraries**

A simple, lightweight desktop application that helps librarians automatically monitor the availability of electronic resources (databases, e-journals, e-books, repositories, websites, etc.).

### About ERMwatch

After more than 20 years working in library systems, I saw a recurring problem: valuable electronic resources often become inaccessible due to broken links or vendor issues, and no one notices until users complain.

ERMwatch was created to solve this practical challenge. It is a **zero-install**, easy-to-use tool designed specifically for librarians.

The application was developed with technical assistance from **Grok (built by xAI)** and **Gemini**. It is released as a **public service** for the global library community.

**Only the executable (.exe) is distributed.** The source code is not shared.

### Key Features

- Automatic monitoring of all e-resources every 10 minutes
- Clear status indicators (Green = Working, Red blinking = Down)
- Desktop notifications when any resource goes down
- Live dashboard with Total, Active, and Down counts
- Easy add/edit/delete resources
- Bulk CSV import/export and template download
- Professional PDF reports with proper formatting
- Support for 8 languages (English, Arabic, Urdu, Chinese, French, German, Spanish, Italian)
- Dark/Light theme toggle
- Automatic desktop shortcut creation
- 100% offline and private (no data collection)
- Basic tamper detection (warns if .exe is modified)

### How to Use

1. Download `ERMwatch.exe` from the Releases section.
2. Place the `.exe` file in any folder 
3. Double-click `ERMwatch.exe` to launch.
   - A desktop shortcut will be created automatically.

4. Add your resources in the **Management** tab.
5. Monitoring starts automatically.
6. Use search, generate PDF reports, import/export CSV, and switch languages/themes as needed.

**No installation required** • **Works completely offline**

### System Requirements

- Windows 10 or 11 (64-bit)
- No Python or additional software needed
