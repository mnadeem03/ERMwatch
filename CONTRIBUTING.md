# Contributing to ERMwatch

Thank you for considering contributing to ERMwatch!

### How Can You Contribute?

We welcome contributions in the following areas:

1. **Bug Reports & Feedback**
   - Report any bugs or unexpected behavior
   - Share suggestions for improvement

2. **New Features**
   - Ideas that would make the tool more useful for libraries

3. **Translations**
   - Help add support for new languages

4. **Code Improvements**
   - Since we distribute only the `.exe`, we welcome pull requests on the source (if made public later)

### Ways to Contribute

- Open an **Issue** on GitHub describing the problem or suggestion
- Send feedback via email or LinkedIn
- Share your experience using ERMwatch with other librarians

### Guidelines

- Be respectful and constructive
- Focus on making the tool better for library staff worldwide
- Remember the project is **strictly non-commercial**
