ERMwatch User Manual
Version 1.1 Simple E-Resources Monitor for Libraries

1. Getting Started
1.	Download ERMwatch.exe from the GitHub Releases.
2.	Place the .exe file in any folder on your computer.
3.	Double-click ERMwatch.exe to launch.
o	A desktop shortcut named ERMwatch will be created automatically on first run.
No installation required.

2. Adding Resources
1.	Go to the Dashboard tab.
2.	Fill in the fields:
o	Type → Select from dropdown (Database, e-Journal, e-Book, Repository, Website, etc.)
o	Resource Name → Clear descriptive name
o	Resource URL → Full URL of the resource
3.	Click REGISTER.
The new resource will immediately appear in the Dashboard tab.

3. Monitoring
•	The app automatically checks all resources every 10 minutes.
•	Green rows = Working
•	Red blinking rows = Down (failure reason shown in the last column)
•	You will receive a desktop notification when any resource goes down.

4. Main Tabs
Dashboard Tab:
•	Live overview with Total, Active, and Down counts
•	Search box to filter resources
•	Color-coded list with status and last checked time
•	Select resources using checkboxes
Management Tab:
•	Open Downtime Log – View history of all downtime events
•	Import Resources (CSV) – Bulk add resources from CSV file
•	Export Resources (CSV) – Save all resources to CSV
•	Download Import Template – Get a blank CSV template

5. Useful Actions
•	Print PDF → Select resources on Dashboard → click PRINT PDF to generate a professional report
•	Language → Change interface language from the top-right dropdown
•	Theme → Toggle between Light and Dark mode using the switch in header

6. Tips for Best Use
•	Use accurate full URLs when adding resources
•	Keep ERMwatch running (you can minimize the window)
•	Check the Downtime Log regularly
•	Use PDF reports for vendor communication or management meetings
•	Export CSV as backup

7. Troubleshooting
•	Resource shows “Down” but opens in browser? Try editing and re-adding the URL.
•	No notifications? Ensure Windows notifications are enabled for ERMwatch.
•	Security Warning appears? Please download the official version from GitHub.
